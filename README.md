# 📖 Bible Adventure — 온가족 성경통독앱

A family-friendly Bible reading app designed to help families 
read through the Bible together — chapter by chapter, day by day.

한 가족이 함께 성경을 통독할 수 있도록 설계된 오픈소스 웹 앱입니다.

---

## ✨ Features / 주요 기능

- 📅 Daily Bible reading tracker / 매일 성경 읽기 진행 체크
- 👨‍👩‍👧‍👦 Family-based progress sharing / 가족 단위 진도 공유
- 📊 Reading progress visualization / 통독 진행률 시각화
- 🔔 Daily reading reminders / 매일 읽기 알림
- 📱 Mobile-friendly responsive design / 모바일 최적화 반응형 디자인

---

## 🎯 Purpose / 프로젝트 목적

This project was created to solve a real problem in family 
and church communities: helping members stay consistent 
with Bible reading plans together.

교회 및 가정에서 성경통독을 꾸준히 이어가기 어렵다는 
현실적인 문제를 해결하기 위해 개발되었습니다. 
목회 현장에서 직접 사용하며 개선하고 있는 실용적인 도구입니다.

## 📌 How to Use / 사용 방법

1. Visit the live demo link above / 위 데모 링크 접속
